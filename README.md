# Faiz Armedi — Portfolio

Personal portfolio website — [armedifaiz.github.io](https://armedifaiz.github.io)

## Quick Start

```bash
npm install
npm run dev        # development server
npm run fetch-all  # fetch external data (photos, badges, projects)
npm run build      # production build
```
